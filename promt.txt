Contexto

Estoy desarrollando una aplicación con Next.js.

Debes integrarte a la arquitectura existente.

Reglas

NO crear nuevas arquitecturas.

NO mover archivos.

NO duplicar lógica.

NO colocar lógica de negocio dentro de componentes visuales.

NO realizar llamadas HTTP directamente desde componentes.

Si necesitas crear nuevos archivos:

Componentes exclusivos del Administrador → app/Administrador/components
Hooks exclusivos del Administrador → app/Administrador/hooks
Utilidades exclusivas del Administrador → app/Administrador/utils
Componentes reutilizables globalmente → src/components
Hooks reutilizables globalmente → src/hooks
Utilidades reutilizables globalmente → src/utils
